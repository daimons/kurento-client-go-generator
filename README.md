# kurento-client-go-generator

这是一个根据 kurento IDL 文件生成的 kurento-golang-client-lib ， 方便 golang 开发者使用。

## 程序运行

### clone submodules (kms-core kms-elements kms-filters)

```
git submodule update --init
``` 

### generator 

```
./make
```

### dest directory

```
./kurento
```

### example

```
./kurento_demo
```
